## Field Object
```
{
    'id': <string>,
    'label': [<string>],
    'type': <object or string>,
    'required': [<boolean>],
    'disabled': [<boolean>]
}
```

## Available Types

### text

``` 'text' ``` or an object like:

```
{
    name: 'text'
}
```

### number

``` 'number' ``` or an object like:

```
{
    name: 'number'
}
```
